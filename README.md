# 📝 Task Manager

A simple Task Manager application built using HTML, CSS, and JavaScript.

## ✨ Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Save tasks using Local Storage
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
