اختصارات لوحة المفاتيح

  • Ctrl + N إنشاء مهمة جديدة
  • Ctrl + F فتح البحث
  • Ctrl + S حفظ التغييرات
  • Delete حذف المهمة المحددة
  • Escape إغلاق النافذة المنبثقة
  • Ctrl + Z تراجع
  • Ctrl + Y إعادة

الأسئلة الشائعة

كيف يمكنني إنشاء مهمة جديدة؟
اضغط على زر "مهمة جديدة" في صفحة جميع المهام، أو استخدم اختصار Ctrl + N.
كيف يمكنني استعادة مهمة محذوفة؟
اذهب إلى "سلة المحذوفات" وابحث عن المهمة، ثم اضغط على زر "استعادة".
كيف أقوم بتغيير الثيم؟
استخدم زر الثيم في أعلى الشاشة، أو من خلال الإعدادات.
